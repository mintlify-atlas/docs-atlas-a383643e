<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-bold dark:text-white text-gray-900 mb-3">Quick Start</h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">Get up and running with GenosDB in minutes</p>
  
  <Steps>
    <Step title="Install GenosDB">
      Install the package via npm, yarn, or pnpm:
      
      <CodeGroup>
      ```bash npm
      npm install genosdb
      ```
      
      ```bash yarn
      yarn add genosdb
      ```
      
      ```bash pnpm
      pnpm add genosdb
      ```
      </CodeGroup>
    </Step>
    
    <Step title="Initialize the database">
      Import and create a new database instance:
      
      ```javascript
      import { gdb } from "genosdb"
      
      const db = await gdb("my-app", { rtc: true })
      ```
      
      <Note>Enable `rtc: true` for real-time P2P synchronization across devices and tabs.</Note>
    </Step>
    
    <Step title="Store and query data">
      Use the graph database API to store nodes and query relationships:
      
      ```javascript
      // Create a node
      const userId = await db.put({ name: "Alice", role: "developer" })
      
      // Query nodes
      const results = await db.map({ 
        query: { role: "developer" },
        $limit: 10 
      })
      
      console.log(results)
      ```
    </Step>
    
    <Step title="Add P2P sync">
      Changes automatically sync in real-time across all connected peers:
      
      ```javascript
      // Subscribe to real-time updates
      db.map({ query: { role: "developer" } }, (data) => {
        console.log("Live update:", data)
      })
      ```
      
      <Tip>Open your app in multiple browser tabs to see real-time synchronization in action!</Tip>
    </Step>
  </Steps>
</div>
